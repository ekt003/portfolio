Project Info

Flocking

Flocking is a cleaned up class project for IGME 202: Interactive Media Development cleaned up. The simulation shows a group of purple spheres that demonstrate the typical flocking AI behavior of cohesion, alignment, and separation. The spheres pursue a random point on the Perlin-Noise generated terrain. Once the flock enters a set radius of a target point, a new point is randomly generated. The simulation is largely passive, however, the user is able to press the space bar to spawn in more flock members at a random location on screen. The user may also press the 'c' key to change the camera view.

Pixel Painter

Pixel Painter is web application made with HTML/CSS/Javascript. It is a grid of pixels that can be painted on to create pixel art. Features include brush sizes, hexcolor/RGB conversions, local saves, and a sample color palette. The project can be found on its own website, while its code is on github.
